## Machine Learning

等同於讓機器找一個函式，將圖片、語音等轉換成一個函式來找出你想找的東西。

深度學習為機器學習中的一種技術。

#### 深度學習中的函式就是一種類神經網路，函式的輸入可以是：

- 向量 vector
- 矩陣 matrix（照片）
- 序列 sequence（一段聲音、文字）

#### 輸出可以是：

- 數字 scalar -> **regression**
- 類別：提供給機器幾個選項叫他選 -> **classification**
- 文字、圖片 -> **產生**
